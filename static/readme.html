  1. Store Name: Gift Print
  2. We are selling: Products with pictures printed on then designed by the costumer.
    The products we print on are : Shirt, Cup, Hat, Teddy Bear, Key Chain and Calender
  3. Additional pages we added:
    • A product design page: the user picks an image to upload, a color for the product and the amount of products to order (and size for shirt).
      The user decides how the image should be located and then finishes editing and adds the product to cart"
    • Admin purchases: The admin can view all the items purchased by users.
      The admin can inspect new orders and get all the needed information to complete the order such as address image to print and information about the products.
      When the admin finishes an order they can mark the order as completed
    • Contact us page: The user can send questions to th admin and receive answers from the admin, just like a chat.
    • Contact users page: The admin receives questions from users. The admin can then choose which user's questions they want to answer.
      Then the admin can send answers to the user, just like a chat.
  4. The things that were hard to do: Writing asynchronous code. Creating the product image with the users design and sending it to the server
